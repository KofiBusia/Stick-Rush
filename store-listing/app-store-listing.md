## Description (4000 chars max)

🏃 STICK RUSH — Ghana's Ultimate Stickman Party Game!

Play 3 epic mini-games, answer educational questions and compete with your friends! Perfect for kids ages 8–16.

3 MINI-GAMES:
• Sprint Dash — Race your stickman! Tap to charge ahead. Beat the CPU to the finish!
• Jump Fever — Endless obstacle runner. Jump, double-jump, collect coins!
• Reflex Blaster — Tap glowing circles before they vanish. Build combos!

LEARN WHILE YOU PLAY:
Answer Mathematics and English questions after each round. Three difficulty levels for ages 8–10, 11–13, and 14–16. Correct answers earn bonus points!

22 COSTUMES TO UNLOCK:
From Common to Mythic rarity — earn costumes by playing more rounds, winning mini-games and answering questions correctly.

MADE FOR WEST AFRICA:
Designed for children across Ghana and West Africa. Educational content aligned with the Ghana Education Service curriculum. Works offline on all devices!

Round 1 is completely FREE. Unlock more rounds for only GHS 5 each.

## iOS BUILD REQUIREMENTS
**IMPORTANT: iOS build requires macOS with Xcode installed.**

Steps to build for iOS (must be done on a Mac):
1. Install Xcode from Mac App Store
2. Install CocoaPods: `sudo gem install cocoapods`
3. Run: `npx cap add ios`
4. Run: `npx cap open ios`
5. In Xcode: set signing team, bundle ID, and version
6. Archive → Distribute App → App Store Connect
7. Submit via App Store Connect at appstoreconnect.apple.com

Apple Developer Account required: $99/year at developer.apple.com
